# 🚀 SamrtHR-Rag-Pipeline: Smart Resume Screener

A robust, AI-powered tool that helps HR teams filter resumes using Context-Aware Search (RAG) and LLMs.

## 🌟 Key Features
* **Semantic Search:** Uses Vector Embeddings (ChromaDB) to find candidates by *meaning*, not just keywords.
* **AI Analysis:** Integrates Llama 3.2 (via Ollama) to summarize and score candidates.
* **Microservices Architecture:** Built with a FastAPI backend and Streamlit frontend.
* **Privacy First:** Runs entirely locally; no data leaves the machine.

## 🛠️ Tech Stack
* **Backend:** FastAPI, Uvicorn
* **Frontend:** Streamlit
* **AI/ML:** ChromaDB, Ollama, PyPDF
* **Language:** Python 3.10+
